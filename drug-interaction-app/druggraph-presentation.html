01

The Problem

  • Adverse drug reactions cause 1.5 million hospitalizations annually in the US alone
  • Approximately 1 in 7 hospitalized patients experience a preventable adverse drug event
  • Many interactions are discovered only after patients experience harmful effects
  • Traditional methods rely on manual literature review, which is impossible to scale
  • Drug combination complexity grows exponentially with each added medication

Core Question: Can we predict drug interactions before they harm patients?

02

Our Solution

  • Model drugs as molecular graphs where nodes are atoms and edges are chemical bonds
  • Train GraphSAGE to learn structural patterns from known interactions
  • Predict interaction probability for any drug pair in real-time
  • Interactive web interface with risk stratification and clinical guidance
  • Supports alias resolution (Aspirin, Advil, Tylenol)

DrugBank provides comprehensive drug information including molecular structures, targets, pathways, and documented drug-drug interactions. Our dataset includes both positive (known interactions) and negative (no reported interaction) examples with a 1:1 negative ratio for balanced training.

04

Graph Construction

  • Nodes: Each atom with features (atomic number, charge, hybridization)
  • Edges: Chemical bonds with features (bond type, stereochemistry)
  • Node Features: Morgan fingerprints (256-bit), drug class, ATC codes
  • Aggregation: Mean pooling via SAGEConv
  • Link Prediction: MLP on concatenated drug embeddings [a, b, a*b]

Training Configuration

  • Learning rate: 0.001 with weight decay 1e-05
  • Batch size: 256 for stable gradients
  • Early stopping with 10 epochs patience
  • 1:1 negative sampling for balanced classes
  • Adam optimizer for adaptive learning
06

GraphSAGE Architecture

  • Encoder: 2-layer DrugGNNEncoder (no BatchNorm)
  • Hidden Dim: 128, Embedding Dim: 64
  • Activation: ELU (between SAGEConv layers)
  • Dropout: 0.3 for regularization
  • Link Predictor: MLP 256 -> 128 -> 1 with dropout (0.2, 0.1)
# Architecture overview DrugGNNEncoder: SAGEConv (256 -> 128) + ELU SAGEConv (128 -> 64) + ELU Dropout (0.3) LinkPredictor: Concat [drug_a, drug_b, a * b] Linear (192 -> 256) + ELU + Dropout(0.2) Linear (256 -> 128) + ELU + Dropout(0.1) Linear (128 -> 1) + Sigmoid

Current Limitations

  • Binary Prediction: Predicts interaction yes/no, not interaction type
  • Pairwise: Evaluates pairs, not full drug cocktails at once
  • Temporal Effects: Does not account for induction time or half-life
  • Dosing Context: Predictions do not include dosage information
  • Novel Interactions: Cannot predict entirely novel interaction types
